Redirige vers un formulaire pour lier une part existante comme fils. Cette action et ces possibilités sont décrites ici
Redirige vers un formulaire de création. La part créée sera liée à la part courante en tant que fils.
Si vous avez les autorisation nécessaire, vous pouvez éditer le fils (mais pas le petit-fils, l'arrière petit-fils, ...) de la part courante.
Il est possible de filtrer les parts affichées en fonction du niveau, de la date et de l'état.
Si la part courante est liée à un Document 3D contenant un fichier STEP, il est possible de générer les parts fils à tout les niveaux. OpenPLM analysera le fichier STEP et affichera les parts générées dans le tableau des résultats. Pour finaliser la création de ces parts, il faudra cliquer sur le bouton Créer qui s'affichera sous le tableau.
Génére le tableau de résultats contenant les parts qui correspondent au critères du filtre.


Vous pouvez utiliser le panneau latéral gauche (Recherche) pour ajouter un fils (une part) à la part courante. Un bouton Ajouter apparait sous les parts pouvant être ajoutées.
Il est aussi possible d'ajouter un fils à la part en remplissant le formulaire.

Le jeu de type, reference, revision doit faire référence à un objet existant.

Même si vous avez utilisé le panneau latéral, vous devez remplir les champs quantité, ordre et unité.
