Welcome to openPLM's documentation!
===================================

How to use this documentation
--------------------------------

If you are an user, you can read :ref:`some tutorials <tutorials>` to start and
:ref:`the plugins documentation <plugins>`.

If you are an administrator, you may read:

    * the installation instructions
    * the available :mod:`settings <settings>`.

If you are a plugins developper, you may want to read :mod:`the api <http_api>`.

If you are a core developper, you may want to read:

    * the :ref:`how-tos <how-tos>`
    * the :ref:`cookbook <cookbook>` which contains some useful pieces of code
    * :ref:`modules' documentation <modules>`
